# prime-reversal-sequence
This script generates a particular [prime reversal sequence](https://www.youtube.com/watch?v=pAMgUB51XZA). 

output: `[1, 0, 0, 0, -2, 2, 0, -6, -6, 6, 0, -4, 4, -10, -14, 10, 4, 14, -30, -42]`
